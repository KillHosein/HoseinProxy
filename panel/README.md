# HoseinProxy Management Panel

یک پنل مدیریت حرفه‌ای برای پروکسی‌های تلگرام با قابلیت مدیریت کانتینرهای داکر.

## ویژگی‌ها
- **مدیریت کامل:** ساخت، حذف، و مدیریت پروکسی‌های MTProto با یک کلیک.
- **مانیتورینگ:** مشاهده لحظه‌ای وضعیت مصرف منابع (CPU, RAM, Disk).
- **رابط کاربری:** طراحی مدرن و واکنش‌گرا (Responsive) با استفاده از Bootstrap.
- **امنیت:** سیستم احراز هویت برای ورود به پنل.

## پیش‌نیازها
- سیستم عامل: Ubuntu 20.04 یا بالاتر (پیشنهادی)
- دسترسی روت (Root)

## نصب و راه‌اندازی سریع

1. فایل‌های پروژه را به سرور خود منتقل کنید.
2. اسکریپت نصب خودکار را اجرا کنید:

```bash
cd HoseinProxy
chmod +x install_panel.sh
sudo ./install_panel.sh
```

3. در حین نصب، نام کاربری و رمز عبور دلخواه برای ورود به پنل را وارد کنید.
4. پس از اتمام، مرورگر خود را باز کرده و آدرس IP سرور را وارد کنید.

## مدیریت دستی (توسعه‌دهندگان)

اگر می‌خواهید پنل را به صورت دستی اجرا کنید:

```bash
cd panel
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

## نکات امنیتی
- پورت 80 به طور پیش‌فرض استفاده می‌شود. برای تغییر پورت، فایل `install_panel.sh` یا سرویس systemd را ویرایش کنید.
- رمز عبورها به صورت Hash شده در دیتابیس `panel.db` ذخیره می‌شوند.

## پشتیبانی از پروتکل‌ها
این نسخه از پروتکل MTProto (نسخه رسمی تلگرام) پشتیبانی می‌کند. در آپدیت‌های بعدی پشتیبانی از SOCKS5 اضافه خواهد شد.
