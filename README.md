# HoseinProxy Management Panel

یک پنل مدیریت حرفه‌ای برای پروکسی‌های تلگرام با قابلیت مدیریت کانتینرهای داکر.
<div dir="rtl">

## ویژگی‌ها
- **مدیریت کامل:** ساخت، حذف، و مدیریت پروکسی‌های MTProto با یک کلیک.
- **مانیتورینگ:** مشاهده لحظه‌ای وضعیت مصرف منابع (CPU, RAM, Disk).
- **رابط کاربری:** طراحی مدرن و واکنش‌گرا (Responsive) با استفاده از Bootstrap.
- **امنیت:** سیستم احراز هویت برای ورود به پنل.
- **گزارش‌گیری:** ثبت دقیق فعالیت‌های مدیران (لاگ).

## پیش‌نیازها
- سیستم عامل: Ubuntu 20.04 یا بالاتر (پیشنهادی)
- دسترسی روت (Root)
- نصب بودن `git` و `curl` (در صورت نصب نبودن، اسکریپت تلاش می‌کند نصب کند)

## نصب و راه‌اندازی سریع

برای نصب، کافیست دستورات زیر را در ترمینال سرور خود اجرا کنید:

```bash
git clone https://github.com/KillHosein/HoseinProxy
cd HoseinProxy
chmod +x install_panel.sh
sudo ./install_panel.sh
```

1. در حین نصب، **نام کاربری** و **رمز عبور** دلخواه برای ورود به پنل را وارد کنید.
2. پس از اتمام نصب، مرورگر خود را باز کرده و آدرس IP سرور را وارد کنید:
   `http://YOUR_SERVER_IP`

## بروزرسانی
برای بروزرسانی پنل به آخرین نسخه:

```bash
cd HoseinProxy
git pull
sudo ./install_panel.sh
```

## مدیریت دستی (توسعه‌دهندگان)

اگر می‌خواهید پنل را به صورت دستی (بدون سرویس systemd) اجرا کنید:

```bash
cd panel
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

## نکات امنیتی
- پورت 80 به طور پیش‌فرض استفاده می‌شود. برای تغییر پورت، فایل `install_panel.sh` یا سرویس systemd را ویرایش کنید.
- رمز عبورها به صورت Hash شده در دیتابیس `panel.db` ذخیره می‌شوند.
