# HoseinProxy Management Panel

یک پنل مدیریت حرفه‌ای و پیشرفته برای پروکسی‌های تلگرام با قابلیت مدیریت کانتینرهای داکر.
<div dir="rtl">

## ویژگی‌های جدید (نسخه 2.0.0)
- **مدیریت سیستم جامع:** اسکریپت `manage.sh` با رابط کاربری گرافیکی (TUI) برای نصب، آپدیت و مدیریت آسان.
- **پشتیبان‌گیری و بازگردانی:** قابلیت تهیه نسخه پشتیبان کامل از دیتابیس و تنظیمات و بازگردانی آن.
- **آپدیت خودکار:** امکان زمان‌بندی به‌روزرسانی خودکار سیستم (Cron Job).
- **بازگشت به نسخه قبل (Rollback):** امکان بازگردانی سریع به نسخه قبلی در صورت بروز مشکل پس از آپدیت.
- **مدیریت وب سیستم:** امکان آپدیت، بکاپ‌گیری و ریستارت سرویس از طریق پنل وب.
- **آمار پیشرفته:** نمودارهای گرافیکی مصرف ترافیک (دانلود/آپلود) در ۷ روز گذشته.
- **تنظیمات اتصال:** قابلیت تنظیم IP و دامنه اختصاصی برای تولید لینک‌های اتصال.
- **تولید لینک هوشمند:** انتخاب بین لینک IP یا دامنه و تولید خودکار سکرت تصادفی.
- **مدیریت پروکسی:** دکمه‌های شروع، توقف، ریستارت و حذف برای هر پروکسی.
- **مانیتورینگ زنده:** مشاهده لحظه‌ای مصرف منابع سرور (CPU, RAM, Disk) با نمودار.

## ویژگی‌های کلی
- **مدیریت کامل:** ساخت، حذف، و مدیریت پروکسی‌های MTProto با یک کلیک.
- **رابط کاربری:** طراحی مدرن و واکنش‌گرا (Responsive) با استفاده از Bootstrap 5.
- **امنیت:** سیستم احراز هویت امن و محافظت در برابر حملات Brute Force (Rate Limiting).
- **گزارش‌گیری:** ثبت دقیق فعالیت‌های مدیران (لاگ).
- **پایداری:** جمع‌آوری خودکار آمار در پس‌زمینه.

## پیش‌نیازها
- سیستم عامل: Ubuntu 20.04 یا بالاتر (پیشنهادی)
- دسترسی روت (Root)
- نصب بودن `git` و `curl` (در صورت نصب نبودن، اسکریپت تلاش می‌کند نصب کند)

## نصب و مدیریت آسان

برای نصب و مدیریت کامل پنل، کافیست مخزن را کلون کرده و اسکریپت مدیریت را اجرا کنید:

```bash
git clone https://github.com/KillHosein/HoseinProxy
cd HoseinProxy
chmod +x manage.sh
./manage.sh
```

پس از اجرای دستور `./manage.sh`، منوی زیر نمایش داده می‌شود:
1. **Install Panel**: نصب کامل پنل و پیش‌نیازها.
2. **Update Panel**: بررسی و دریافت آخرین نسخه.
3. **Uninstall Panel**: حذف کامل پنل (با امکان بکاپ‌گیری و نمایش فایل‌های حذفی).
4. **Restart Service**: ریستارت کردن سرویس پنل.
5. **View Logs**: مشاهده لاگ‌های سیستم.
6. **Schedule Updates**: فعال/غیرفعال کردن آپدیت خودکار شبانه.
7. **Backup Data**: تهیه نسخه پشتیبان دستی.
8. **Rollback Version**: بازگشت به نسخه قبلی.

### دسترسی به پنل وب
پس از نصب موفقیت‌آمیز، مرورگر خود را باز کرده و آدرس IP سرور را وارد کنید:
`http://YOUR_SERVER_IP`

## نکات امنیتی
- پورت 80 به طور پیش‌فرض استفاده می‌شود.
- رمز عبورها به صورت Hash شده ذخیره می‌شوند.
- سیستم دارای محدودیت تلاش برای ورود (Rate Limiting) است.
- پیشنهاد می‌شود از دامنه‌های دارای SSL (در صورت تنظیم Nginx) استفاده کنید.
